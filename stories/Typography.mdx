# Typography

## Ciznel

This font should only be used in the following scenarios:

- Titles
- Buttons
- Links, in headers, isolated from body content
- Large display text

This font should almost always be used with the `--text-font-weight-normal` variable.

## Newsreader

This font should be used for all readable text:

- Article text
- Card content
- List items
- Links, inline with other body text
